# Intent

Last updated: 2026-03-12
Task status: completed

## Change Intent

Add the smallest useful artifact drawer behavior to the public surface so that the artifact pack feels like an active review boundary, not a passive section.

## Must Preserve

- the page must remain static and lightweight
- the existing visual language must remain calm and editorial
- the drawer behavior must stay understandable without introducing app-like complexity
- the artifact pack must remain compact and readable

## Must Not Introduce

- a frontend framework
- hidden network behavior
- fake product complexity
- a drawer interaction that overwhelms mobile

## Success Condition

A user can move from the first review scene to the artifact pack and clearly see the required review artifacts as the minimal boundary.
