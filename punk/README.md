# punk

Spec-driven development toolkit — keep contracts and implementation in sync, with jj and git support.

## Install

```sh
cargo install --path punk-cli
```

## Usage

```sh
punk init       # initialize a punk workspace
punk plan       # generate an implementation plan from a contract
punk check      # check implementation against the active contract
punk receipt    # record a receipt for the completed contract
punk status     # show current workspace status
punk config     # manage punk configuration
```
